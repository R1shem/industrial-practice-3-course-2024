# industrial-practice-3-course-2024
Учебный проект по предмету производственная практика выполненный на первом семестре третьего курса.

WPF-приложение, использующее базу данных, созданной с помощью MySQL через утилиту MySQL WorkBench. Диаграммы бизнес-процессов

Функционал: вывод, удаление и запись различных информации из базы данных. Вывод в Excel. Основной упор в приложении был на ограничения вводимых данных
